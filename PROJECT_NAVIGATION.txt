НАВИГАЦИЯ ПО ПРОЕКТУ БОТА
=========================

1) Точка входа и конфиг
-----------------------
- app.py
  Запуск бота, подключение роутеров, старт polling.

- config.py
  Чтение переменных окружения: BOT_TOKEN, ADMIN_ID, DB_PATH.

- .env
  Ваши реальные настройки запуска.


2) База данных
--------------
- bot/database/db.py
  Вся работа с SQLite:
  - создание таблиц
  - стартовые тексты/FAQ/настройки
  - users, applications, texts, faq, settings, images
  - методы чтения/обновления данных

- data/example_data.sql
  Пример SQL-схемы и тестовых данных.


3) Пользовательская логика
--------------------------
- bot/handlers/user.py
  Команды и сценарии пользователя:
  - /start, /menu
  - главное меню
  - разделы (установка, регистрация, карточка)
  - FAQ
  - поддержка
  - FSM-заявка
  - отправка заявки админу


4) Админ-логика
---------------
- bot/handlers/admin.py
  Админ-панель:
  - /admin
  - просмотр заявок
  - изменение приветствия
  - изменение текстов разделов
  - изменение FAQ
  - изменение контакта менеджера
  - загрузка изображений (включая приветствие)
  - ответ пользователю по кнопке


5) Кнопки и состояния
---------------------
- bot/keyboards/inline.py
  Все inline-кнопки (кнопки в сообщениях).

- bot/states/forms.py
  FSM-состояния:
  - заявка пользователя
  - редактирование в админке
  - ответ пользователю


6) Константы и утилиты
----------------------
- bot/utils/constants.py
  Тексты кнопок и ключи разделов/FAQ для callback-логики.

- bot/utils/formatters.py
  Вспомогательные функции (например, нормализация username).


7) Обработка ошибок
-------------------
- bot/handlers/errors.py
  Глобальный обработчик исключений.


8) Зависимости
--------------
- requirements.txt
  aiogram, aiosqlite, python-dotenv.
